#  Bilibili 简单复刻版

这是一个模仿 **Bilibili** 的复刻项目，主要用于学习与练手。  
项目实现了常见的视频播放平台功能，包括视频获取、视频播放、本地收藏、用户登录等模块。  

## 功能实现

-  **视频获取**：完成了首页推荐视频信息获取的 api 对接   
-  **收藏功能**：可以自定义收藏夹，并进行视频收藏管理，利用coredata进行本地视频存储
-  **搜索功能**：模糊查找视频信息，显示搜索列表并支持跳转
-  **界面还原**：  尽可能还原 Bilibili 的视觉风格
